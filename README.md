# English Mastery - 12 Design Variants

A comprehensive collection of 12 modern landing page designs for an English Course website. Built with vanilla HTML/CSS, Tailwind CDN, and Google Fonts.

---

### 🇮🇩 Deskripsi (Bahasa Indonesia)
Proyek ini berisi 12 varian desain landing page untuk kursus bahasa Inggris, mulai dari gaya minimalis, editorial, hingga neo-brutalisme. Setiap desain dioptimalkan untuk konversi dan estetika modern.

**Fitur Utama:**
- 12 Varian Desain (Notion, Stripe, Apple, Linear, dll.)
- Gallery Comparison Page
- Responsive Design
- Integrasi WhatsApp Registration
- Tanpa Build Step (Vanilla HTML/Tailwind CDN)

---

### 🇺🇸 Description (English)
This project features 12 distinct landing page design variants for an English language course, ranging from minimalist and editorial to neo-brutalist styles. Each design is optimized for conversion and modern aesthetics.

**Key Features:**
- 12 Design Variants (Notion, Stripe, Apple, Linear, etc.)
- Gallery Comparison Page
- Responsive Design
- WhatsApp Registration Integration
- No Build Step (Vanilla HTML/Tailwind CDN)

### 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/ihsanbagus/english-course-designs.git
   ```
2. Open `gallery.html` in your browser to compare all designs.
3. Use any `index-*.html` file as your starting point.
